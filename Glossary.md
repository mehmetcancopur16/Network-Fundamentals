# 📖 Glossary of Networking Terms

A comprehensive dictionary of essential networking terminology used throughout this repository. Understanding these terms is foundational for any cybersecurity professional or network engineer.

---

* **802.11:** The IEEE standard for wireless local area networks (Wi-Fi).
* **AES (Advanced Encryption Standard):** A highly secure encryption algorithm used in WPA2 and WPA3 wireless security.
* **ARP (Address Resolution Protocol):** The protocol used to map an IP address to a physical MAC address on a local network.
* **Bandwidth:** The maximum throughput capacity of a network medium.
* **BSSID (Basic Service Set Identifier):** The MAC address of the wireless access point (router) broadcasting a specific Wi-Fi network.
* **Collision Domain:** A network segment where data collisions can occur. Switches create separate collision domains for each port.
* **DNS (Domain Name System):** The protocol used to resolve human-readable hostnames (like `google.com`) into IP addresses.
* **Encapsulation:** The process where a layer adds its specific header information to the data payload received from the layer above it.
* **Firewall:** A network security device that monitors and controls incoming and outgoing traffic based on predetermined security rules.
* **Hop Count:** A routing metric that counts the number of routers a problem must pass through to reach its destination.
* **IDS (Intrusion Detection System):** A passive security appliance that monitors network traffic for malicious activity and generates alerts.
* **IPS (Intrusion Prevention System):** An active security appliance that sits inline with traffic and can block or drop malicious packets in real-time.
* **IPv4:** The 4th version of the Internet Protocol, using 32-bit addresses (e.g., `192.168.1.1`).
* **IPv6:** The 6th version of the Internet Protocol, using 128-bit hexadecimal addresses to solve the IPv4 exhaustion problem.
* **MAC Address:** A 48-bit physical hardware address burned into a Network Interface Card (NIC).
* **NAT (Network Address Translation):** A mechanism that translates private IP addresses into a public IP address, allowing local devices to access the internet.
* **NDP (Neighbor Discovery Protocol):** The IPv6 equivalent of ARP. It uses multicast instead of broadcast to find other devices on the local link.
* **OSI Reference Model:** A conceptual 7-layer model defining the functions of a telecommunication or computing system.
* **Proxy Server:** An intermediary server that handles client requests. A forward proxy protects the client, while a reverse proxy protects the server.
* **SAE (Simultaneous Authentication of Equals):** The secure key establishment protocol used in WPA3 to prevent offline dictionary attacks.
* **SSID (Service Set Identifier):** The human-readable name of a wireless network.
* **Stateful Firewall:** A firewall that remembers the state of active connections and makes decisions based on the context of the traffic flow.
* **Subnet Mask:** A value used to divide an IP address into a network portion and a host portion.
* **TCP (Transmission Control Protocol):** A connection-oriented, reliable Transport layer protocol.
* **UDP (User Datagram Protocol):** A connectionless, fast Transport layer protocol that does not guarantee data delivery.
* **VLAN (Virtual LAN):** A logical grouping of network devices into a single broadcast domain, isolating traffic at Layer 2.
* **VPN (Virtual Private Network):** A secure, encrypted tunnel over an untrusted network (like the Internet).
* **WPA2 / WPA3:** Wi-Fi Protected Access protocols used to secure wireless networks. WPA3 is the latest and most secure iteration.
